Bella Italia

This example map of Italy showcases some ways you can style Kartograph maps. The base colors are defined in CSS. Then a glow filter is applied to, image textures and SVG filters.

Also you can see heavy use of map symbols here, namely labels and icons. The labels are set in the free web font Aquiline, created by Manfred Klein.
