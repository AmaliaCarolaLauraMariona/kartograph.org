These maps demonstrates the map symbol clustering techniques built into Kartograph.js. They all show the location of UK cities sized by the number of people that have visited this website in 2012.

no clustering — The first map shows the +500 symbols without clustering. London (2k visits) is completely covered by cities with only a few visitors.

k-means — Symbols are combined using k-Means clustering. Inspired by and uses implementation released along with this Polymaps example.

noverlap — Here simply all overlapping symbols are combined, as described here. The name is inspired by a Gephi plugin.

See documentation for details how to use symbol clustering.
