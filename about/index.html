The magic behind Kartograph

The core concept of Kartograph is to separate the mapping process in the map generation and map rendering part. The client-side library kartograph.js renders pre-generated maps stored in SVG files. You can chose among hundrets of ready-to-use maps or simply generate your own using kartograph.py, the open source Python SVG map generator.

The Kartograph framework essentially consists of two major parts. The first is a Python library to generate SVG maps from various geodata sources (shapefiles, KML, GeoJson). By 'generate' we mean that the geodata is filtered, projected, generalized, joined, cropped and finally stored as vector graphics.

The second part is a JavaScript library that will load and render the SVG map in any web browser.

Kartograph is written in Python and JavaScript (CoffeeScript, acutally).
