What is Kartograph?

Kartograph is a simple and lightweight framework for building interactive map applications without Google Maps or any other mapping service. It was created with the needs of designers and data journalists in mind.

Actually, Kartograph is two libraries. One generates beautiful SVG maps; the other helps you to create interactive map applications that runs across all major browsers.

Kartograph.py

A powerful Python library for generating beautiful, Illustrator-friendly SVG maps.

  • Renders SVG maps from shapefiles and PostGIS
  • Compact using Visvalingam simplification
  • Stores metadata in data attributes
  • Built on top of GDAL/OGR and shapely
  • Licensed under AGPL

Read the Docs   Fork on Github

Kartograph.js

A JavaScript library for creating interactive map applications on top of SVG maps.

  • Data vis friendly
  • Runs 100% stand-alone. No tile-server needed.
  • Gracefully degrades to Internet Explorer 7+
  • Built on top of Raphaël and jQuery
  • Licensed under LGPL

Current Progress

At the moment Kartograph is still in an early pre-release stadium, which means that it basically works and is already used a lot by its author but there are some more things left to do to make it as easy-to-use for everybody else. This includes tutorials and step-by-step examples and API documentation.

If you like Kartograph and can’t wait to support its development, please find the donation buttons on the left. Any contribution is much appreciated. Thanks!
