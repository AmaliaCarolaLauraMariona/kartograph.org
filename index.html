Why Kartograph?

Kartograph is a new framework for building interactive map applications without the need for Google Maps or any other mapping service. It was created on top of RaphaelJS with the needs of designers and data journalists in mind. In short, for people who want to tell data-driven stories using maps.

Please check out the showcase section to see what Kartograph can do for you or dive into the technical details to learn more about it.

Current Progress

At the moment Kartograph is in an alpha-release stadium, which means that it basically works and is already used a lot by its author but there are some more things left to do to make it as easy-to-use for everybody else.

If you like Kartograph and can’t wait to support its development, please find the donation buttons on the left. Any contribution is much appreciated. Thanks!

Features

  • Supporting beautiful maps™
