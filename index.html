A free library for creating beautiful vector maps using SVG.

svgmap is a free library for thematic mapping beyond tile-based Mercator maps. The main idea behind svgmap is to divide the mapping process in two parts.

At first, the Python-side of svgmap (svgmap.py) generates SVG map from high quality Natural Earth shapefiles. The SVG maps contain information on the used map projection as well as metadata for every displayed geometry, like country-codes. It allows you to literally control every parameter to get excactly the map that you want.

The second part is a light-weight JavaScript library (svgmap.js) which loads the SVG maps and allows you to make them interactive as well as to connect it with your datasets. The appearance of the maps is highly customizable using CSS. To display the maps, no server-side components are needed, so you're completely indepentend. It's just you, your SVG map and your data.
